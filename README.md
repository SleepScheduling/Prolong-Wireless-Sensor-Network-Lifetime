# Prolong-Wireless-Sensor-Network-Lifetime
Hybrid Meta-Heuristic Optimization Approach for Sleep Scheduling Mechanism to Prolong Wireless Sensor Network Lifetime

Run Main.m file to show the Results

Dataset Description:
  The dataset is not a publicly downloaded dataset.
  It is a simulation-generated dataset created using:
      * MATLAB 2020a
      * Wireless Sensor Network (WSN) simulation environment
  It includes:
      * Node energy values
      * Packet Delivery Ratio (PDR)
      * Network lifetime metrics
      * Coverage and redundancy parameters
  Data is generated under controlled experimental conditions defined in the paper.

  Summary : 
      This Work optimizes wireless sensor networks by dynamically scheduling nodes between sleep and active states using a hybrid optimization algorithm, thereby reducing energy consumption and extending network lifetime.
